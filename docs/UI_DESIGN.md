# CCooler - C盘清理工具 UI设计（极简版）

## 设计理念
- 极简主义，专注清理功能
- 一屏完成所有操作
- 无多余页面和弹窗
- 快速扫描，一键清理

- 危险: #EF4444 (红色)
- 背景: #F9FAFB (浅灰)
- 文字: #111827 (深灰)

## 界面元素

### 1. 标题栏
- 应用名称 "CCooler"
- 最小化、关闭按钮
- 可拖动区域

### 2. 左侧导航栏
- 宽度: 120px
- 背景色: #FFFFFF
- 三个菜单项：
  - **🧹 C盘清理** - 默认选中
  - **📊 软件统计** - 显示已安装软件及占用空间
  - **💬 微信迁移** - 微信数据迁移专用页面
- 选中状态：蓝色背景 + 左侧蓝色竖条

### 3. 磁盘状态（C盘清理页面）
- 进度条显示C盘使用情况
- 显示已使用/总容量

### 4. 清理项列表（C盘清理页面）
8个清理项，每项显示：
- 复选框（安全项默认选中，危险项默认不选）
- 清理项名称
- 可清理大小

**清理项包括：**

**安全清理项（默认选中）：**
1. **系统临时文件** - %TEMP%, Windows\Temp, Prefetch
2. **浏览器缓存** - Chrome, Edge, Firefox缓存和Cookie
3. **回收站** - 清空回收站
4. **Windows更新缓存** - SoftwareDistribution, Windows.old
5. **系统文件清理** - 对应Windows磁盘清理的"清理系统文件"功能
   - Windows错误报告
   - Windows Defender扫描历史
   - 缩略图缓存
   - 传递优化文件

**需用户确认项（默认不选）：**
6. **下载目录** - 用户下载文件夹
   - 显示文件列表供用户选择性删除
7. **应用缓存** - 常见应用缓存目录

### 5. 统计信息（C盘清理页面）
- 显示总可清理空间

### 6. 操作按钮（C盘清理页面）
- **开始扫描** - 扫描所有清理项
- **立即清理** - 清理已选中的项目

## 功能详细说明

### 页面1: C盘清理

#### 系统临时文件
- `%TEMP%` - 用户临时文件
- `C:\Windows\Temp` - 系统临时文件
- `C:\Windows\Prefetch` - 预读取文件

#### 浏览器缓存
- Chrome: `%LOCALAPPDATA%\Google\Chrome\User Data\Default\Cache`
- Edge: `%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Cache`
- Firefox: `%APPDATA%\Mozilla\Firefox\Profiles\*.default\cache2`

#### 回收站
- 清空所有驱动器的回收站

#### Windows更新缓存
- `C:\Windows\SoftwareDistribution\Download`
- `C:\Windows.old` (如果存在)

#### 系统文件清理
需要管理员权限：
- Windows错误报告: `C:\ProgramData\Microsoft\Windows\WER`
- Windows Defender扫描历史
- 缩略图缓存数据库
- 传递优化文件: `C:\Windows\SoftwareDistribution\DeliveryOptimization`

#### 下载目录
- 扫描 `C:\Users\用户名\Downloads`
- 显示文件列表（按大小排序）
- 用户可选择性删除

#### 应用缓存
- 常见应用的缓存目录
- `%LOCALAPPDATA%\Temp`

### 页面2: 软件统计

#### 功能
- 读取注册表获取已安装软件列表
  - `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall`
  - `HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall`
- 显示软件名称、安装路径、占用大小
- 按大小排序
- 统计总数量和总占用空间

#### 注意事项
- 只读功能，不提供卸载
- 计算文件夹大小可能较慢，显示加载状态

### 页面3: 微信迁移

#### 功能
- 自动检测微信安装路径
  - 注册表: `HKEY_CURRENT_USER\Software\Tencent\WeChat`
  - 默认路径: `C:\Program Files\Tencent\WeChat`
- 扫描微信数据目录
  - 默认: `C:\Users\用户名\Documents\WeChat Files`
- 分类统计：聊天记录、文件、图片/视频、其他数据
- 提供迁移引导步骤
- "打开微信"按钮：启动微信程序
- "重新扫描"按钮：刷新数据统计

#### 注意事项
- 不直接操作微信数据，避免数据损坏
- 引导用户使用微信官方迁移功能
- 如果微信未安装，显示提示信息
