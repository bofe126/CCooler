# CCooler - C盘清理工具 UI设计（极简版）

## 设计理念
- 极简主义，专注清理功能
- 每个功能一屏完成，无需跳转
- 无多余弹窗
- 快速扫描，一键清理

## 配色方案
- 主色: #3B82F6 (蓝色)
- 危险: #EF4444 (红色)
- 背景: #F9FAFB (浅灰)
- 文字: #111827 (深灰)

## 界面元素

### 1. 标题栏
- 应用名称 "CCooler"
- 最小化、关闭按钮
- 可拖动区域

### 2. 左侧导航栏
- 宽度: 120px
- 背景色: #FFFFFF
- 三个菜单项：
  - **🧹 C盘清理** - 默认选中
  - **📊 软件统计** - 显示已安装软件及占用空间
  - **💬 微信迁移** - 微信数据迁移专用页面
- 选中状态：蓝色背景 + 左侧蓝色竖条

### 3. 磁盘状态（C盘清理页面）
- 进度条显示C盘使用情况
- 显示可用空间/总容量（如：可用 105.0 GB / 共 300 GB）
- 颜色逻辑（类似 Windows 系统）：
  - 使用率 < 80%: 蓝色（正常）
  - 使用率 >= 80%: 黄色（警告）+ 提示"磁盘空间不足，建议清理"
  - 使用率 >= 90%: 红色（严重）+ 提示"磁盘空间严重不足，建议立即清理"

### 4. 清理项列表（C盘清理页面）
7个清理项，每项显示：
- 复选框（安全项默认选中，危险项默认不选）
- 清理项名称
- 可清理大小
- "查看详情"文本按钮（点击展开详情面板）

**清理项包括：**

**安全清理项（默认选中）：**
1. **系统临时文件** - %TEMP%, Windows\Temp, Prefetch
2. **浏览器缓存** - Chrome, Edge, Firefox缓存和Cookie
3. **回收站** - 清空回收站
4. **Windows更新缓存** - SoftwareDistribution, Windows.old
5. **系统文件清理** - 对应Windows磁盘清理的"清理系统文件"功能
   - Windows错误报告
   - Windows Defender扫描历史
   - 缩略图缓存
   - 传递优化文件

**需用户确认项（默认不选）：**
6. **下载目录** - 用户下载文件夹
   - 勾选后将清空整个下载目录
7. **应用缓存** - 常见应用缓存目录

### 5. 统计信息（C盘清理页面）
- 显示总可清理空间（仅统计已选中的清理项）

### 6. 操作按钮（C盘清理页面）
- **开始扫描** - 扫描所有清理项
- **立即清理** - 清理已选中的项目

### 详情面板交互说明

1. **打开方式**：
   - 点击清理项右侧的"查看详情"文本按钮
   - 详情面板从右侧滑入动画（300ms）

2. **关闭方式**：
   - 点击详情面板右上角的 [ × ] 按钮
   - 点击详情面板外的区域（遮罩层）
   - 按 ESC 键

3. **内容区域**：
   - 顶部：清理项名称 + 关闭按钮
   - 说明区：清理项的功能说明
   - 统计区：总大小、文件数等统计信息
   - 详细列表：包含的路径、文件、子项等
   - 底部：安全提示或警告信息

4. **状态标识**：
   - ✅ 安全项：绿色提示
   - ⚠️ 需确认项：黄色提示
   - 🚨 危险项：红色警告

5. **响应式设计**：
   - 详情面板宽度：300px
   - 主列表区域自动收窄
   - 保持整体布局平衡

## 状态图标说明

### 清理项状态图标
- **✓** - 已完成（绿色）
- **🔄** - 进行中（蓝色动画）
- **⏳** - 等待中（灰色）
- **❌** - 失败/错误（红色）
- **☑** - 已选中（蓝色复选框）
- **☐** - 未选中（灰色复选框）

### 页面状态图标
- **🔍** - 扫描中
- **🗑️** - 清理中
- **✅** - 成功完成
- **⚠️** - 警告/部分完成
- **💡** - 提示信息
- **🎉** - 庆祝/完美状态

### 进度条样式
- **已完成部分**：蓝色填充 `████████████`
- **未完成部分**：灰色填充 `░░░░░░░░░░░░`
- **百分比显示**：右侧显示具体百分比

### 按钮状态
- **正常**：蓝色背景，白色文字
- **禁用**：灰色背景，灰色文字，标注 `(禁用)`
- **危险操作**：红色背景（如"取消清理"）
- **次要操作**：白色背景，蓝色边框（如"重新扫描"）

## 交互行为说明

### C盘清理页面
1. **初始加载**：显示未扫描状态，"立即清理"按钮禁用
2. **点击"开始扫描"**：
   - 按钮变为"取消扫描"
   - 逐项扫描，显示实时进度
   - 扫描完成后按钮恢复为"重新扫描"，"立即清理"按钮启用
3. **勾选/取消清理项**：
   - 实时更新"可清理"统计
   - 至少选中一项时"立即清理"按钮才可用
4. **点击"立即清理"**：
   - 显示确认对话框（如果选中了危险项）
   - 开始清理，显示总进度和当前项
   - 可随时取消，已清理的不会恢复
5. **清理完成**：
   - 显示清理结果和明细
   - 更新C盘空间进度条
   - 点击"完成"返回初始状态

### 软件统计页面
1. **首次进入**：自动开始扫描
2. **扫描过程**：显示已找到软件数量和计算进度
3. **扫描完成**：显示软件列表，可滚动查看
4. **点击"刷新列表"**：重新扫描

### 微信迁移页面
1. **首次进入**：自动检测微信
2. **检测到微信**：自动扫描数据目录
3. **扫描完成**：显示数据分类统计
4. **点击"打开微信"**：启动微信程序
5. **点击"重新扫描"**：刷新数据统计

## 功能详细说明

### 页面1: C盘清理

#### 系统临时文件
- `%TEMP%` - 用户临时文件
- `C:\Windows\Temp` - 系统临时文件
- `C:\Windows\Prefetch` - 预读取文件

#### 浏览器缓存
- Chrome: `%LOCALAPPDATA%\Google\Chrome\User Data\Default\Cache`
- Edge: `%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Cache`
- Firefox: `%APPDATA%\Mozilla\Firefox\Profiles\*.default\cache2`

#### 回收站
- 清空所有驱动器的回收站

#### Windows更新缓存
- `C:\Windows\SoftwareDistribution\Download`
- `C:\Windows.old` (如果存在)

#### 系统文件清理
需要管理员权限：
- Windows错误报告: `C:\ProgramData\Microsoft\Windows\WER`
- Windows Defender扫描历史
- 缩略图缓存数据库
- 传递优化文件: `C:\Windows\SoftwareDistribution\DeliveryOptimization`

#### 下载目录
- 扫描 `C:\Users\用户名\Downloads`
- 勾选后将清空整个下载目录
- 建议用户在清理前手动检查重要文件

#### 应用缓存
- 常见应用的缓存目录
- `%LOCALAPPDATA%\Temp`

### 页面2: 软件统计

#### 功能
- 读取注册表获取已安装软件列表
  - `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall`
  - `HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall`
- 显示软件名称、安装路径、占用大小
- 按大小排序
- 统计总数量和总占用空间

#### 注意事项
- 只读功能，不提供卸载
- 计算文件夹大小可能较慢，显示加载状态

### 页面3: 微信迁移

#### 功能
- 自动检测微信安装路径
  - 注册表: `HKEY_CURRENT_USER\Software\Tencent\WeChat`
  - 默认路径: `C:\Program Files\Tencent\WeChat`
- 扫描微信数据目录
  - 默认: `C:\Users\用户名\Documents\WeChat Files`
- 分类统计：聊天记录、文件、图片/视频、其他数据
- 提供迁移引导步骤
- "打开微信"按钮：启动微信程序
- "重新扫描"按钮：刷新数据统计

#### 注意事项
- 不直接操作微信数据，避免数据损坏
- 引导用户使用微信官方迁移功能
- 如果微信未安装，显示提示信息
